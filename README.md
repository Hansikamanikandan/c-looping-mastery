## 📖 Overview

**C Looping Mastery** is a curated collection of **15 C programs** that demonstrate the practical use of looping statements to solve mathematical, logical, and pattern-based problems.

The repository focuses on strengthening programming fundamentals through iterative problem-solving using clean, structured, and beginner-friendly C code.

## ▶️ Getting Started

Compile any program using GCC:

```bash
gcc filename.c -o output
```

Example:

```bash
gcc 03_factorial_of_a_number.c -o factorial
```

Run the executable:

**Windows**
```bash
factorial.exe
```

**Linux/macOS**
```bash
./factorial
```
